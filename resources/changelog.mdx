## Latest Releases

### Version 1.2.0

**Released:** January 22, 2026

<AccordionGroup>
<Accordion title="Features">

**Dictionary Updates:**
- Integrated and revised STPhrases/TWPhrases dictionaries
- Added medical terminology entries
- Added Nvidia brand name translations
- Added Japanese rabbit character variants (兔/兎)
- Comprehensive test coverage improvements

**Dictionary Format:**
- Introduced comment syntax for `.txt` dictionaries
- New sorting rules for dictionary files
- `TWPhrasesRev` now maintained directly in repository with consistency tests

**Test Infrastructure:**
- Migrated test cases to JSON format for better maintainability
- Improved test validation and error reporting

</Accordion>

<Accordion title="Bug Fixes">

- Fixed `tw2sp`/`s2twp` configuration errors causing incorrect conversions
- Fixed UTF-8 truncation boundary overflow read issue
- Corrected multiple dictionary entries and phrase conversions

</Accordion>

<Accordion title="Build & Dependencies">

- Upgraded Bazel to 8.5.1
- Upgraded to C++17 with marisa-trie 0.3.0
- Updated `nan` dependency for Node.js compatibility
- Fixed GCC 15 and C++17 deprecation warnings

</Accordion>

<Accordion title="Platform Support">

- Added macOS 14 support
- Removed Python 2 remnants
- Registered hermetic Python 3.12 toolchain
- Updated Node.js test matrix, removed Node 18 and Windows x86
- Improved CI artifact uploads and task cancellation

</Accordion>

<Accordion title="Documentation">

- Added Traditional Chinese contributing guide (CONTRIBUTING.md)
- Updated README with Bazel testing instructions
- Added contributor acknowledgments
- Listed opencc-wasm in third-party implementations
- Fixed PyPI release scripts

</Accordion>
</AccordionGroup>

**GitHub:** [View on GitHub](https://github.com/BYVoid/OpenCC/releases/tag/ver.1.2.0)

### Version 1.1.0

**Released:** May 10, 2020

<Warning>
**Major Release**: Introduces new `.ocd2` dictionary format with significant file size reductions.
</Warning>

<AccordionGroup>
<Accordion title="Dictionary Format">

**New `.ocd2` Format:**
- Based on Marisa Trie 0.2.5
- Massive size reduction: `STPhrases.ocd` (4.3MB) → `STPhrases.ocd2` (924KB)
- Faster loading times
- Improved memory efficiency

</Accordion>

<Accordion title="Dependencies">

- Upgraded rapidjson to 1.1.0
- Upgraded tclap to 1.2.2
- Upgraded googletest to 1.11.0

</Accordion>

<Accordion title="Dictionary Updates">

- Changed default conversion for 涌/湧
- Corrected multiple phrase conversions
- Improved accuracy across all conversion types

</Accordion>

<Accordion title="Distribution">

- Provided Windows precompiled binaries (AppVeyor)
- Added benchmark results for performance comparison

</Accordion>
</AccordionGroup>
